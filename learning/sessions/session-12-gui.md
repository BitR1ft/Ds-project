# Session 12: GUI Application Walkthrough

**Duration:** 45-60 minutes  
**Goal:** Understand the graphical interface

## Features
- Folder browser for custom scanning
- Network traffic monitoring
- Real-time threat dashboard
- Scan history
- Configuration panel

## Technology
- Python Tkinter (cross-platform)
- Threading for background tasks
- Queue for inter-thread communication

## User Flow
1. Browse → Select folder
2. Start Scan → Background thread
3. Monitor → Real-time updates
4. Results → Multi-tab display
5. Reports → Save for later
