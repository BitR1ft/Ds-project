# Code Examples

This directory contains runnable code examples for each learning session.

## Structure

Each example corresponds to a session and demonstrates key concepts:

- **simple-trie.cpp** - Basic Trie implementation (Session 3)
- **basic-graph.cpp** - Simple graph operations (Session 5)
- **avl-demo.cpp** - AVL tree rotations (Session 7)
- **heap-example.cpp** - Heap operations (Session 8)
- **hash-table-demo.cpp** - Hash table with chaining (Session 9)
- **linked-list.cpp** - Linked list operations (Session 10)

## How to Use

1. **Compile examples:**
   ```bash
   g++ -std=c++17 simple-trie.cpp -o trie_demo
   ```

2. **Run:**
   ```bash
   ./trie_demo
   ```

3. **Modify and experiment:**
   - Change input data
   - Add features
   - Test edge cases
   - Break things and fix them!

## Learning Approach

- **Read the code** with comments
- **Run the example** to see output
- **Modify the code** to test understanding
- **Complete exercises** based on examples
